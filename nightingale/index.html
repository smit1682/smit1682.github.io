How to format the paste
  • Begin every question with Q:
  • Options as A) B) C) D) — two to six per question
  • Ans: takes the correct letter, or the full option text
  • Exp: is the explanation shown during review
  • Leave a blank line between questions
  • Optional per question: Topic: and Diff: (easy / medium / hard) — anything without them takes the batch settings above
  • A JSON array works too, if the questions already live in one
